# KidMath - Worksheet Generator
## This is a personal learning project created for educational purposes and to explore different code concepts with ai concepts. 

* **Status:** Personal sandbox / Portfolio piece.
* **License:** This project is open-source and available for public educational use under the MIT License.
* **Purpose:** Academic research and technical skill development.

KidMath is a clean, minimalist web application designed to generate printable math worksheets for children. It focuses on high-contrast black and white aesthetics, making it perfect for printing on standard paper.

## Features

- **Dynamic Math Problems**: Generate random addition and subtraction problems.
- **Customizable Difficulty**: 
    - The first number is always two digits (10-99).
    - The second number can be set to **Single Digit** (1-9), **Double Digit** (10-99), or **Mixed**.
- **Visual Aids**: Each math problem includes a number line (arrow) visual aid to help kids solve the exercises.
- **Multi-Page Support**: Batch generate from 1 to 50 pages at once.
- **Print Optimized**: Specifically designed to fit 4 panels per page on standard **US Letter** paper, with browser headers and footers hidden for a professional look.
- **Simple Controls**: Easy-to-use interface to switch modes, randomize problems, and configure page counts.

## How to Use

1. **Select Mode**: Choose between "Addition", "Subtraction", or "Mixed".
2. **Configure Digits**: Use the "2nd Num" dropdown to set the difficulty for the second number.
3. **Set Pages**: Enter the number of unique pages you wish to generate.
4. **Randomize**: Click the **↻ Randomize** button to generate new problems.
5. **Print**: Click the **Print** button to open the browser's print dialog. The layout is pre-configured to look perfect on a single sheet of paper.

## Technical Details

- Built with **Vanilla HTML5, CSS3, and JavaScript**.
- Uses the **Fredoka** Google Font for a friendly, child-appropriate look.
- Zero dependencies; runs directly in any modern web browser.
